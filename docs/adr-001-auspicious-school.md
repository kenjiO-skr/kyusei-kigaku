# ADR-001 吉方位判定の採用流派

- 状態：**確定**（2026年5月29日／令和8年5月29日）
- 決定者：本人（プロジェクトオーナー）
- 関連：引き継ぎ仕様 §9-3「吉方位判定の採用流派の確定」、仕様メモ §4

## 背景
九星気学の吉方位判定は流派差が大きく、特に①退気（自分が生む五行）の扱い、②自分が剋す五行（死気）の扱い、③定位対冲の採否で割れる。仕様 §5 の年盤検証データ（2026年・一白中宮・二黒土星視点）がテストオラクルであり、流派の選択はこれに直結する。

## §5 が暗黙に固定している判断（飛泊式で逆算・検証済み）
2026年盤を飛泊式で再現し、§5 の各方位判定と突き合わせた結果：

| 関係（対二黒・土） | 該当星 | §5判定 | 含意 |
|---|---|---|---|
| 生気（火→土） | 九紫(南東) | 凶※本命的殺で打消 | 凶殺非該当なら本来は最大吉 |
| 比和（土） | 八白(東) | **吉** | 比和＝吉 |
| 退気（土→金） | 七赤(南西) | **中立（退気）** | 退気は吉に入れない |
| 殺気（木→土） | 四緑/三碧(北東/西) | 小凶 | 殺気＝小凶 |
| 死気（土→水） | 一白(中宮) | 例なし | 今年は判定不能 |

さらに **定位対冲**は2026年盤で三碧＠西の1件のみ該当するが、§5 は西を「**小凶（木剋土）**」とし「凶」に格上げしていない。→ **§5 は定位対冲を凶殺として採用していない**。

## 決定
**流派A：園田系・標準**を採用する。

- **吉**＝凶殺非該当 かつ 回座星が 二黒に対し **生気（火＝九紫）** または **比和（土＝二黒/八白）**
- **退気（金＝六白/七赤）** … 中立（吉に含めない）
- **殺気（木＝三碧/四緑）** … 小凶
- **死気（水＝一白）** … 小凶（§5に例がないため園田系に準拠して小凶とする）
- **定位対冲** … 不採用
- **小児殺** … 不採用（自分用）
- 凶殺：五黄殺・暗剣殺・歳破/月破/日破・本命殺・本命的殺・八方塞がり

## 帰結
- 流派A は §5 を**改訂なしで完全再現**でき、テストオラクルに直結する。
- 流派B（退気も弱吉）・流派C（定位対冲も凶）は §5 改訂が必要なため**不採用**。
- 本決定の変更は、この ADR の改訂と `docs/verification-data.md` のテストオラクル再生成を前提とする。
