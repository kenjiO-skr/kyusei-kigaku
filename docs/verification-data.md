# 検証データ（テストオラクル）

実装が満たすべき正解データ。これが赤になったら**実装が誤り**。出典：引き継ぎ仕様 §5。

## 1. 飛泊（盤配置）— 全盤共通
式：`配置星 = ((d − 1 + (k − 5)) を9で正の剰余) + 1`（k=中宮の星, d=後天定位星, 結果1〜9）

後天定位盤（五黄中宮, k=5）：
```
南東 4 | 南  9 | 南西 2
東  3 | 中  5 | 西  7
北東 8 | 北  1 | 北西 6
```

検証：**全九星 k=1..9 を中宮にしたとき、各宮の配置が一意に決まること**。
代表例 k=1（一白中宮）の配置：
| 方位 | 定位星d | 配置星 |
|---|---|---|
| 中宮 | 5 | 一白(1) |
| 北 | 1 | 六白(6) |
| 北東 | 8 | 四緑(4) |
| 東 | 3 | 八白(8) |
| 南東 | 4 | 九紫(9) |
| 南 | 9 | 五黄(5) |
| 南西 | 2 | 七赤(7) |
| 西 | 7 | 三碧(3) |
| 北西 | 6 | 二黒(2) |

## 2. 2026年・年盤（一白中宮）／本命星=二黒土星 視点
2026年＝丙午年（午＝南）。中宮=一白。下表どおりの盤・吉凶を返すこと。

| 方位 | 回座する九星 | 二黒視点の判定 | 理由 |
|---|---|---|---|
| 中宮 | 一白 | — | — |
| 北 | 六白 | **凶** | 暗剣殺（五黄=南の対冲）＋歳破（午=南の対冲=子=北） |
| 北東 | 四緑 | 小凶 | 木が土を剋す（殺気） |
| 東 | 八白 | **吉** | 土の比和 |
| 南東 | 九紫 | **凶** | 本命的殺（二黒=北西の対冲） |
| 南 | 五黄 | **凶** | 五黄殺 |
| 南西 | 七赤 | 中立（退気） | 土が金を生む（退気） |
| 西 | 三碧 | 小凶 | 木が土を剋す（殺気） |
| 北西 | 二黒 | **凶** | 本命殺（本命星の回座方位） |

注：これは**年盤のみ**の単体検証。実アプリの「今日の吉方位」は日盤・月盤の重なりで確定する。日盤・月盤の正解データは暦計算実装後（Phase 2-3）に別途生成し本ファイルに追記する。

## 3. 本命星 立春境界
- 1971年生まれ → **二黒土星**（本人・回帰テストに固定）
- 2月初旬（立春前日以前）生まれ → **前年**で算出（1星ズレ）。年により立春は 2/3〜2/5 で揺れる。

## 4. 暦境界（Phase 2 で確定。出典・検証ログは `docs/calendar-data-source.md`）
確定値は `src/calc/calendar-data.js`。A体（天文計算）×B体（国立天文台公表値転記）で全点一致。

### 立春（本命星の年境界）— テストオラクル
- **2/3 になる年**：2021・2025・2029・2033（これ以外で 2/3 を返したら誤り）
- **2/5 になる年**：1951・1952・1956・1960・1964・1968・1972・1976・1980・1984
- **上記以外は全て 2/4**（1950–2035）。回帰固定：**1971=2/4**。
- 境界時刻が JST 0時直前/直後の薄いケース（実装が日付を取り違えやすい）：
  - 2021立春 2/3 23:58 → **2/3**／2025立春 2/3 23:09 → **2/3**／1984立春 2/5 00:21 → **2/5**

### 二至（日盤の陰遁陽遁基準, 2025–2035）— テストオラクル
- 夏至：全年 **6/21**。冬至：2028・2029・2032・2033 が **12/21**、他は **12/22**。
- 極限ケース：**2025冬至 = 12/22**（計算上 00:00:21 だが NAOJ 確定値も 12/22）。

### 60干支日（甲子位置）— テストオラクル
- **甲子 ⟺ (JDN mod 60) === 11**（Fliegel–Van Flandern 整数JDN・正午基準）。基準 2024-01-01=甲子。
- 検算固定点：2000-01-01=戊午、2024-03-01/2024-12-26/2026-02-19/2026-12-16=甲子。

### 日盤（陰遁陽遁＋閏）— テストオラクル（Phase 3b 確定）
算出規則・出典は `docs/calendar-data-source.md`。実装は `calendar.js` の `dayBoard`／`dayBoardCenter`。
オラクル＝yakumoin.info 公表日盤（`/daily/direction/date/YYYYMMDD`, 2000–2050）＋ nobml 計算例。

規則：冬至直近の甲子=一白で陽遁(+1)、夏至直近の甲子=九紫で陰遁(−1)。
**九星の閏**＝前の切替から240日間隔になる回だけ、甲子を30日前倒しして甲午とし（間隔210日）、
冬至=七赤／夏至=三碧で開始（切替前後で同星が1日重複）。

通常期（中宮の星）:
- 2026-02-04=一白（陽遁）／2026-06-21=七赤（陰遁）／2028-08-15=四緑／2030-07-01=三碧／2035-03-10=四緑

2031年冬の閏（甲午→七赤・陽遁。**12/19 と 12/20 が七赤で2日連続＝閏の重複**）:
- 2031-12-19=七赤／2031-12-20=七赤／2031-12-22=九紫／2031-12-23=一白／2031-12-31=九紫
- 2032-01-12=三碧／2032-01-18=九紫／2032-01-19=一白／2032-01-20=二黒

注：対象範囲は SOLSTICES（2025–2035）に依存。最初の切替（2025夏至付近）より前は範囲外。
1950–2035の範囲で閏は 2031冬のみ（2020夏・2043夏は範囲外）。
